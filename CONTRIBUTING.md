# Contributing

## Branch Naming

All branches must include the ticket ID:

```
HDR-##
```

**Examples:**
- `HDR-01`
- `HDR-42`
- `HDR-123`

## Commit Messages

All commits must follow this format:

```
[HDR-##] <message>
```

**Examples:**
- `[HDR-01] Add user authentication endpoint`
- `[HDR-42] Fix login redirect issue`
- `[HDR-123] Update dashboard layout`

## Pull Requests

1. Create a branch from `main` using the naming convention above
2. Make your changes with properly formatted commits
3. Open a PR back to `main`
4. Ensure all checks pass before requesting review
